# TikTok AI Video Workflow

## Purpose
Turn TikTok reels into a repeatable weekly system using:
- KlingAI for video generation
- Suno for music generation
- CapCut for stitching, captions, timing, and final packaging

The goal is consistency, not novelty.

## Core principle
Do not reinvent the process every week.

The workflow should feel like a small production line:
1. choose idea
2. write script
3. generate visuals
4. generate/select music
5. finish in CapCut
6. publish
7. log results

## Default weekly cadence
### Monday
- choose 1 topic
- confirm hook, insight, CTA
- finalize 45–60 second script

### Tuesday
- generate first KlingAI video draft
- review for visual clarity and pacing

### Wednesday
- generate or choose Suno music
- refine Kling output only if needed

### Thursday
- assemble in CapCut
- add captions
- add title text
- tighten pacing
- export final version

### Friday
- publish
- log what worked
- move next ideas forward in backlog

## Standard production sequence
### Step 1: choose the idea
Use the backlog.
Pick one idea that has:
- clear audience
- clear hook
- low production friction
- strong relevance right now

### Step 2: write the script
Use a simple structure:
- Hook
- Problem
- Insight
- Example
- CTA

Target length:
- 100 to 140 spoken words
- 45 to 60 seconds

### Step 3: generate video in KlingAI
Input:
- final script or visual scene prompt
- fixed style prompt
- vertical 9:16 requirement
- pacing requirement
- tone requirement

Output:
- one first-draft visual asset

### Step 4: generate/select music in Suno
Use music only if it helps the video.
The music should support the message, not overpower it.

Output:
- one selected track or concept track

### Step 5: finish in CapCut
CapCut is the packaging layer.
Do:
- trim dead space
- align scenes to narration
- add captions
- add headline text
- set audio levels
- export vertical final

### Step 6: publish
Post the video.
Then log:
- topic
- hook used
- style used
- time spent
- what worked
- what should improve next week

## Operating rules
- One primary format at a time
- One editor at a time
- One visual style family at a time
- One music style family at a time
- One weekly video minimum

## Initial recommended format
Start with:
- faceless AI-generated visual reel
- strong hook text
- clear narration or text-led structure
- subtle background music
- clean fast captions

This is easier to repeat than trying to make every reel feel cinematic.

## Quality gate before publish
A video should pass these checks:
- strong hook in first 2 seconds
- one clear message only
- visuals support the message
- captions are readable
- music is not distracting
- pacing is tight
- CTA is simple

## Failure modes to avoid
- too many different styles
- prompt tweaking for hours
- weak script hidden under pretty visuals
- generic AI voice/tone
- too much editing for low-value videos
- skipping the backlog and starting from zero each week

## Minimum viable success condition
Post 1 good-enough reel per week for 12 weeks.

That matters more than making 1 spectacular reel and disappearing.
