# Suno Prompt Templates

## Music rules
Use music to support the content, not dominate it.
The best short-form background music is:
- subtle
- rhythmic
- modern
- emotionally aligned with the content
- easy to sit under voice/captions

## Template 1 - Clean modern background
Create a subtle modern instrumental track for a short-form video.
Mood: focused, clean, confident.
Use light rhythmic movement and a polished digital texture.
Avoid vocals that compete with narration.

## Template 2 - Motivating but not cheesy
Create an instrumental short-form background track.
Mood: energetic, forward-moving, inspiring, but not cheesy.
Suitable for TikTok explainers and practical lesson videos.
Avoid cinematic excess and avoid overpowering the speaker.

## Template 3 - Calm intelligent tone
Create a minimal instrumental track for a short-form educational video.
Mood: calm, smart, modern, slightly ambient.
Keep the arrangement simple and suitable for spoken content layered on top.

## Template 4 - Builder/operator tone
Create a modern instrumental for builder/operator style content.
Mood: productive, focused, steady momentum.
Should feel like progress and execution, not hype.
Suitable for software, systems, planning, and workflow-related videos.

## Selection rule
If the music draws attention away from the message, it is the wrong track.
