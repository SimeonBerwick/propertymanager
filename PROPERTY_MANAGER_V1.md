# Property Manager App - Streamlined V1

## Product framing
The easiest way for small landlords to manage maintenance requests and keep everyone in sync.

## Core user
- small landlords
- small property managers
- owner-operators managing a limited portfolio

## Problem to solve first
Maintenance coordination is chaotic when it lives across texts, calls, email, and memory.

## V1 scope
### Core workflows
- tenant issue submission
- photo upload
- category + urgency
- landlord/operator inbox
- request status tracking
  - new
  - scheduled
  - in progress
  - done
- vendor dispatch
- tenant updates / communication trail
- property / unit issue history
- basic reporting
  - open vs closed
  - aging
  - repeat issue flags

## Why this wedge
- high-frequency pain
- easy to understand
- immediately useful
- expandable later
- much cleaner than trying to replace full property management suites on day one

## Success test
A small landlord should be able to move maintenance coordination out of ad hoc texting and into one simple tracked workflow.
