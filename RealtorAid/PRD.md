# Realtor Aid v1 PRD snapshot

## Product truth for this build
A realtor needs a simple operating cockpit for leads, not a bloated CRM.

The first useful version must let an agent:
- capture leads quickly
- see priority queues immediately
- open any lead and understand context fast
- log touches without friction
- schedule the next follow-up before context is lost

## Required v1 scope
- schema/data model
- lead list
- lead detail
- quick add lead
- activity logging
- follow-up scheduling
- dashboard queues: New / Due Today / Overdue / Stale
- demo seed data

## Explicit non-goals for v1
- auth
- MLS integrations
- email sync
- SMS sync
- complex automations
- team permissions
- production database infrastructure
