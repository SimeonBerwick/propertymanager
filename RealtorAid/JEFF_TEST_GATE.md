# Jeff Test Gate

## Goal of first finished build
Jeff should be able to:
1. log in
2. upload an empty-room photo
3. choose a room type and style
4. submit a staging job
5. wait for processing
6. review multiple variants
7. download the selected result

## Pass criteria
- upload is stable
- job status is understandable
- at least one variant is returned successfully
- UI is simple enough to use without explanation
- failure states are clear

## Fail criteria
- upload flow breaks
- jobs disappear or stall silently
- results are not tied clearly to the source photo
- download/export is confusing or absent
- output quality is so poor that the workflow cannot be judged
