# TikTok Script Templates

## Template 1 - Practical lesson
- Hook: Most people get [topic] wrong because [reason].
- Problem: They focus on [wrong thing], which causes [pain].
- Insight: The real move is [core insight].
- Example: For example, [simple real example].
- CTA: Follow for more practical systems like this.

## Template 2 - Contrarian take
- Hook: The usual advice on [topic] is backwards.
- Problem: People are told to [common advice], but that creates [problem].
- Insight: A better approach is [alternative].
- Example: Here’s what that looks like: [example].
- CTA: If you want, I’ll break this into a part two.

## Template 3 - Build in public
- Hook: Here’s what I’m learning while building [thing].
- Problem: I assumed [assumption], but ran into [friction].
- Insight: The better way is [lesson].
- Example: In practice, I changed [specific thing].
- CTA: Follow if you want more real build lessons.

## Template 4 - Simple framework
- Hook: If you’re stuck on [topic], use this 3-step framework.
- Problem: Most people overcomplicate it.
- Insight: Step 1 is [x], step 2 is [y], step 3 is [z].
- Example: Here’s how that works in real life: [example].
- CTA: Save this if you want to use it later.

## Template 5 - Mistake / lesson
- Hook: The fastest way to fail at [topic] is [mistake].
- Problem: It feels smart at first, but leads to [bad outcome].
- Insight: The better move is [better behavior].
- Example: I learned this when [example].
- CTA: Follow for more lessons that save time.
