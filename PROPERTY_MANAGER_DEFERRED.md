# Property Manager App - Deferred Features

These are intentionally excluded from the streamlined V1.

## Deferred from V1
- rent collection
- accounting / bookkeeping
- lease management
- e-signatures
- tenant screening
- owner financial reporting portal
- broad vendor marketplace
- enterprise permissions / workflow layers
- mobile apps for every user role if web is enough initially
- broad AI automation beyond clearly useful workflow help

## Why defer
These features add complexity, compliance burden, and build drag.
They should only come after the maintenance command center proves real value.

## Future expansion path
If V1 works, expand in this order:
1. inspections
2. richer vendor workflows
3. document storage
4. owner reporting
5. rent / billing features
