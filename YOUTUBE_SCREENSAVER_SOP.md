# YouTube Screensaver Video SOP

## Goal
Produce 2–10 hour ambient/screensaver videos cheaply, cleanly, and with low licensing risk.

## Cheapest viable stack
- **DaVinci Resolve** — main editing
- **FFmpeg** — extending loops / batching exports
- **Canva** — thumbnails, channel art, simple text overlays
- **Pexels** — free stock video
- **Pixabay** — free stock video/audio
- **YouTube Audio Library** — safe music source
- **Google Drive** — asset storage
- **YouTube Studio** — upload, metadata, scheduling

## Accounts / websites needed
- YouTube / YouTube Studio
- Pexels
- Pixabay
- Canva
- Google Drive

Optional later:
- paid stock library
- paid music library
- more cloud storage
- lightweight SEO tool

## Licensing / compliance checklist
Before using any asset:
- save source URL
- record license type
- save creator name if required
- note whether attribution is needed
- store proof in a license log

Avoid:
- ripped music
- random “royalty-free” YouTube uploads
- unclear AI music rights
- footage with visible trademarks/logos if rights are uncertain

## Production workflow
1. Pick one ambience type
   - rain
   - fireplace
   - ocean
   - forest
   - night ambience
2. Gather looping-safe footage
3. Gather properly licensed music/ambience
4. Save all assets into organized folders
5. Record licenses in a log
6. Build one master timeline in DaVinci Resolve
7. Use FFmpeg if needed to extend loops cleanly
8. Export target durations:
   - 2 hour
   - 4 hour
   - 8 hour
9. Create thumbnail in Canva
10. Upload via YouTube Studio
11. Add title, description, tags, and playlist placement
12. Keep a reusable template so each new upload is not a brand-new project

## What to avoid
- multiple paid subscriptions too early
- custom editing from scratch for every video
- overdesigned motion graphics for commodity content
- asset chaos without license records
- trying to force growth with ad spend too early
- chasing 4K perfection before proving traction
