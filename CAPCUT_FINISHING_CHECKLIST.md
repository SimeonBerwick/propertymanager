# CapCut Finishing Checklist

## Before editing
- confirm final script
- confirm final visual assets
- confirm selected music track

## Edit pass
- trim dead space
- align scene changes to the script
- remove awkward pauses
- tighten pacing

## Captions
- add readable captions
- keep caption style consistent
- ensure captions do not block key visuals
- fix obvious auto-caption errors

## Title / hook text
- add opening headline if needed
- make first 2 seconds visually clear
- do not overload the screen with text

## Audio
- narration/voice should be clear
- background music should sit underneath, not compete
- reduce music volume if clarity suffers

## Visual consistency
- same aspect ratio: vertical 9:16
- similar look and pacing across videos
- no random font/style changes unless intentional

## Final export check
- watch from start to finish once
- hook is clear fast
- one message only
- CTA present
- no obvious typos
- no broken visuals
- export final vertical version

## Publish log
After export, log:
- title/topic
- hook used
- script template used
- Kling prompt used
- Suno track used
- publish date
- notes for next time
